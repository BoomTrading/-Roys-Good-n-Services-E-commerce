Cannot Delete Room

This room cannot be deleted because it has active bookings in the system.

Instead of deleting rooms, consider marking them as unavailable if they need to be taken out of service.

To manage this room's availability:

  1. Go back to the rooms list
  2. Click "Edit" on this room
  3. Uncheck the "Is Available" checkbox
  4. Save the changes
